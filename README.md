# NoCH
 NoCH is an independent multiplayer game.
 Technologies used:
 HTML5 Canvas,
 Node JS,
 Matter.js
 
 ## Server
### Quickstart
```shell
$ git clone git@github.com:DmitryTsyganov/NoCH.git
$ cd NoCH/server
$ npm install
$ git clone -b master --single-branch https://github.com/liabru/matter-js node_modules/matter-js
$ node server.js
```
